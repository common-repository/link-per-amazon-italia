== Description ==

Mostra bottoni con link alle offerte di Amazon Italia per suggerire prodotti correlati al post. Specifica un "campo personalizzato" nel post corrente per mostrare un bottone cliccabile ai tuoi visitatori e portarli ad una pagina con offerte correlate su Amazon.it

OPZIONI DI VISUALIZZAZIONE

a) SHORTCODE
Usa lo shortcode [Link_Amazon] per mostrare il bottone in specifici post o pagine.

Se preferisci mostrarlo nei file PHP del template utilizza <?php echo do_shortcode('[Link_Amazon]'); ?>


b) WIDGET
Trascina il widget "Link_Amazon" nella sidebar o altre posizioni personalizzate usando il menu Aspetto > Widget
Puoi specificare un titolo personalizzato e dove mostrare il widget (sempre, articoli, archivi o pagine).


IMPOSTAZIONI
Visita la pagina di amministrazione di Link_Amazon per cambiare le impostazioni.
Puoi scegliere un colore personalizzato il per il bottone, il tuo Amazon affiliate tag e il testo da mostrare nel bottone.
Ricorda di aggiungere al post il "Campo Personalizzato" chiamato "prodotto" e scrivi il nome del prodotto nel campo "VALORE".


== Installazione ==

Per attivare il plugin, vai in Plugins > Aggiungi > Carica Plugin e seleziona il file .zip del plugin.

Poi premi ATTIVA per installare il plugin correttamente.
